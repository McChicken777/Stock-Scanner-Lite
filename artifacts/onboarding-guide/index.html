Chapter 0
Before You Start
Read this first. It explains how the app is structured and what order to set things up in.

What is Stock Scanner Lite?

Stock Scanner Lite is a web-based workshop management app. It tracks:

  • Jobs (Work Orders) — each customer job broken into steps that workers complete one by one
  • Materials/Stock — raw materials and purchased parts, with automatic deduction as jobs progress
  • Attendance — clock-in/out, leave requests, live attendance view
  • Purchasing — reorder alerts, purchase orders to suppliers

User Roles — Who Sees What

RoleCan seeCannot see
Admin Everything — all pages, all jobs, all settings —
Supervisor Jobs, queues, attendance, stock-take, reports Admin settings, user management
Worker My Tasks, Station Queues, Attendance Jobs list, stock, settings, reports

Logging In

1
Open the app in a browser

Navigate to the URL provided by your account manager. The login page will appear automatically.

Screenshot: Login page
2
Enter your admin credentials

Your initial admin username and password are provided in your welcome email. Enter them and click Log In.

✅
First login Use the admin account for all setup steps. You can change the password later in Admin → Users.
3
You land on the Jobs page

After login, admins are redirected to the Jobs (Work Orders) page. The left sidebar gives access to all sections. Start with Admin → Company.

Chapter 1
Company Settings
Navigate to Admin → Company in the sidebar. Fill in every section before moving to the next chapter.
1
Company Name & Timezone

In the Company Information card, enter your company name exactly as it should appear on reports and printed tags.

Set Timezone from the dropdown. This controls when "today" starts for attendance and reports. Incorrect timezone = wrong attendance dates.

🖼️
Screenshot: Company name and timezone selector
2
Working Hours

In the Work Hours card, set the default start and end time for your working day (e.g. 07:00 – 16:00). This is used for attendance alerts when workers forget to clock out.

⚠️
Important: Set this before workers start clocking in. It cannot be back-dated.
3
Shifts (optional)

If your workshop runs multiple shifts (e.g. morning / afternoon), click Add Shift and enter each shift name, start time, and end time.

Workers assigned to a shift will have their attendance checked against those hours instead of the default working hours.

🖼️
Screenshot: Shifts section with example morning/afternoon shifts
4
Leave Policy

Set the number of annual leave days each worker receives per year (e.g. 20). This is used when workers submit leave requests — the system automatically tracks remaining days.

✅
Tip: You can override the leave allowance per individual worker later in Admin → Users.
5
Save

Click Save Changes at the bottom of each card. There is no auto-save — if you leave the page without saving, changes are lost.

Chapter 2
Roles
Navigate to Admin → Roles. Roles control which station queues a worker can see and work on. Create roles before adding workers.

Think of roles as job titles in your workshop. Examples: Cutter, Painter, Welder, Assembler, Foreman. A worker can have only one role, but a station can accept multiple roles.

🖼️
Screenshot: Admin → Roles page showing role list
1
Create a Role

Click New Role (top right). Enter a name (e.g. "Painter") and click Save.

Repeat for every job type in your workshop. Common roles:

  • Cutter / CNC Operator
  • Painter / Powder Coater
  • Welder / Fabricator
  • Assembler
  • Foreman / Supervisor (can also be the isSupervisor flag on the user)
2
Understand how roles connect to stations

When you create a Station (Chapter 4), you will assign one or more roles to it. Only workers with those roles will see that station's queue on their mobile device.

ℹ️
Example: Station "Paint Booth" → assigned roles: Painter, Powder Coater. Workers with either role see the Paint Booth queue. Workers with role "Cutter" do not.
Chapter 3
Workers & Users
Navigate to Admin → Users. Add every person who will use the app — workers, supervisors, and any additional admins.
1
Add a Worker

Click Add User. Fill in:

  • Name — full name as it appears in reports
  • Username — what they type to log in (no spaces, e.g. "jsmith")
  • Password — set a temporary password; the worker can change it after first login
  • Role — select the role you created in Chapter 2
  • Is Supervisor — tick this if the worker should see the supervisor views (attendance live, stock-take, reports) even though their role is "worker"
🖼️
Screenshot: Add User dialog with fields filled in
2
Set Leave Allowance (optional override)

After saving, click the user's row to edit them. If this worker has a different annual leave allowance than the company default, enter it in Leave Days.

3
Assign to a Shift (optional)

If you created shifts in Chapter 1, select the worker's shift from the Shift dropdown. Leave blank to use default working hours.

4
Repeat for all workers

Add every person before moving on. You can always add more workers later — the setup order only matters for roles and stations.

✅
Tip: Workers do not need email addresses. Username + password is all they need to log in from any device on the workshop floor.

Chapter 4
Production Stations
Navigate to Admin → Stations. Define every physical workstation or process stage in your workshop.

A station is a named step-type that job steps are assigned to. When a job step is at "Paint Booth", only painters see it in their queue. This keeps each worker's screen uncluttered — they only see the work meant for them.

🖼️
Screenshot: Admin → Stations page showing station list
1
Create a Station

Click Add Station. Enter:

  • Station Name — e.g. "CNC Cutting", "Paint Booth", "Assembly", "Welding"
  • Roles — select one or more roles that are allowed to work at this station
  • Color — optional color-coding shown in the job detail view
🖼️
Screenshot: Add Station dialog
2
Create all stations

Think about every distinct stage a job passes through in your workshop. Common examples:

  • Design / Drawing
  • Material Prep / Cutting
  • CNC / Laser
  • Welding / Fabrication
  • Grinding / Finishing
  • Paint / Powder Coat
  • Assembly
  • Quality Check
  • Packing / Dispatch
⚠️
Create all stations now. Template steps reference stations — if you forget one, you'll need to go back and add it before building that template step.
Chapter 5
Stock & Materials
This chapter sets up your physical locations, your material products, and enters the opening stock quantities. Use Admin → Suppliers, Locations, and Products from the sidebar.

Step A — Add Suppliers

Navigate to Admin → Suppliers. Add every company you buy materials from.

1
Create a Supplier

Click New Supplier. Enter the supplier name, email, phone, and any notes (e.g. lead time, account number). Click Save.

Repeat for all suppliers. You can always add more later.

Step B — Add Storage Locations

Navigate to Locations in the sidebar. A location is a physical place where stock is stored — e.g. "Main Warehouse", "Shelf A1", "Paint Store".

2
Create Locations

Click Add Location. Enter a name. You must have at least one location before you can enter stock quantities.

✅
Simple setup: If you have one storage area, create a single location called "Warehouse". You can always add more locations later as your tracking needs grow.

Step C — Add Products / Materials

Navigate to Products in the sidebar. A product can be a raw material, a purchased part, or a finished goods item.

3
Create a Material Product

Click New Product. Fill in:

  • Name — e.g. "Steel Tube 50×50×3mm"
  • SKU — your internal code (optional)
  • Category — groups products in the stock-take view
  • Item Type — select Purchase for anything you buy from a supplier
  • Supplier — select from your supplier list
  • Unit Cost — cost per unit (used for valuation reports)
  • Buffer Stock — the quantity that triggers a low-stock alert (e.g. 10)
  • Target Stock — ideal quantity to reorder up to (e.g. 50)
🖼️
Screenshot: New Product form filled in
4
Enter Opening Stock Quantities

After creating the product, go to Work → Stock-Take in the sidebar. This is the fastest way to enter all your starting quantities at once.

For each material, type the current physical count in the Counted box. Items where the count differs from the system total are highlighted orange. Click Save when done.

🖼️
Screenshot: Stock-Take page with counts being entered
✅
Alternatively: Open a product, navigate to its detail page, and use the "Adjust Stock" button to add stock to a specific location.
5
Repeat for all materials

Add every material, consumable, and purchased part your workshop uses. These are referenced when building template steps (Chapter 6) — a step can automatically deduct material when completed.

Chapter 6
Job Templates
Navigate to Work → Templates. Templates are reusable step-by-step recipes for your common job types. Every new job is cloned from a template.

Think of a template as a checklist for a job type — e.g. "Steel Gate", "Powder Coat Only", "Aluminium Window". When you create a job, you pick a template and all its steps are copied in automatically.

🖼️
Screenshot: Work → Templates page showing template list
1
Create a Template

Click New Template. Enter a name (e.g. "Steel Security Gate") and an optional description. Click Create.

2
Open the Template Outline Editor

Click the template name to open it. You'll see the Outline Editor — a hierarchical step builder. Steps can be nested (parent step → sub-steps).

🖼️
Screenshot: Template outline editor with steps visible
3
Add Steps

Click + Add Step (or press Enter at the end of a step line in the editor). For each step, fill in:

  • Step Name — what the worker sees, e.g. "Cut all tube sections to length"
  • Station — which workstation this step belongs to (e.g. CNC Cutting). Workers at that station will see this step in their queue.
  • Estimated Time — how long the step should take (used for scheduling reports)
  • Consumes Material — optional. Select a product and quantity that will be automatically deducted from stock when the step is completed.
  • Notes — instructions, tolerances, or any info the worker needs
🖼️
Screenshot: Step edit panel with Station and Consumes Material fields
4
Build out the full workflow

Add all steps in the order workers should complete them. A typical steel gate template might look like:

  1. Design & Drawing Design
  2. Cut frame sections CNC Cutting — consumes: Steel Tube × qty
  3. Cut infill bars CNC Cutting
  4. Weld frame Welding
  5. Weld infill Welding
  6. Grind welds Grinding
  7. Sandblast Finishing
  8. Powder coat Paint Booth
  9. Fit hinges & handles Assembly
  10. Quality check QC
5
Create all your common job types

Build a template for every recurring job type. You can always clone a template (click the copy icon) to create a variation without rebuilding from scratch.

⚠️
Changing a template does not update existing jobs. Jobs already created are independent copies. Template changes only affect new jobs created after the change.
Chapter 7
Creating Your First Job
Navigate to Work → Jobs. A job (work order) is a specific customer order that workers will complete step by step.
1
Create a New Job

Click New Job (top right of the Jobs page). Fill in:

  • Job Name — e.g. "Smith Residence — Security Gate"
  • Template — select the template that matches this job type (e.g. "Steel Security Gate"). All steps are imported automatically.
  • Customer — optionally link to a customer record
  • Due Date — the deadline. Overdue jobs appear with a red badge and the admin attention counter will increase.
  • Priority — Normal, High, or Urgent. Affects the ordering in queues.
  • Notes — any job-specific information (measurements, colour, special instructions)
🖼️
Screenshot: New Job form with template selected
2
Review the Job Detail

After saving, you are taken to the job detail page. All steps from the template are listed. Each step shows its station, status (Not Started → In Progress → Done), and assigned worker.

🖼️
Screenshot: Job detail page with all steps listed
3
Adjust Steps if Needed

Steps can be edited, reordered, or deleted directly in the job (this only affects this job, not the template). Add job-specific steps by clicking Add Step.

4
The Job is Now Live

Workers at each station will see the relevant steps appear in their queue as soon as the job is created. No further action from the admin is needed — workers claim and complete steps themselves.

✅
Print Tag: Use the print icon on the job detail page to print a physical tag/label for the job. Stick it to the workpiece so it travels through the shop with the job.
Chapter 8
Daily Worker Workflow
Workers use the app on any phone or tablet. Their view is simplified — they only see what's relevant to their role.

Morning: Clocking In

1
Open the app and log in

Worker opens the app URL in their phone browser, enters username and password. The app loads their simplified view with a bottom navigation bar.

🖼️
Screenshot: Worker login and bottom nav (My Tasks / Queues / Attendance)
2
Clock In via Attendance

Tap Attendance in the bottom nav. Tap Clock In. The system records the time. The button changes to Clock Out.

✅
Reminder: Workers must clock in every day. The admin/supervisor can view who is clocked in on the live attendance board (Admin → Attendance → Live).

During the Day: Working on Jobs

3
Check the Queue

Tap Queues in the bottom nav. The worker sees all steps assigned to their station(s) that are not yet done. Steps are ordered by job priority and due date.

🖼️
Screenshot: Station queue on mobile with step cards
4
Claim a Step

Tap a step card to expand it. Tap Claim to assign it to yourself. The step moves to the top of your queue with a green highlight. Other workers can see it is claimed and will skip it.

5
Complete the Step

Once the physical work is done, tap the large Done button on the claimed step card. The step is marked complete. If the step consumes a material, stock is automatically deducted.

🖼️
Screenshot: Claimed step card with green border and Done button
6
My Tasks view

Workers can also tap My Tasks to see only steps they have personally claimed across all jobs. This is useful when a worker has claimed steps in multiple jobs.

End of Day: Clocking Out

7
Clock Out

Tap Attendance → Clock Out. Total hours for the day are recorded.

⚠️
Forgot to clock out? The supervisor or admin can correct attendance records from Admin → Attendance → Report.

Submitting Leave Requests

8
Submit a Leave Request

In the Attendance tab, scroll down to Leave Requests. Tap New Request. Choose the type (Annual Leave, Sick Leave, etc.), the dates, and add a reason. Tap Submit.

The admin receives the request in Admin → Leave Inbox and can approve or reject it.

Chapter 9
Supervisor & Admin Workflow
Supervisors and admins have a wider view of the app. This chapter covers the most important daily and periodic tasks.

Monitoring Jobs

1
Jobs Page Overview

Navigate to Work → Jobs. Jobs are grouped by status (In Progress, Pending, Completed). Use the search bar to find a specific job by name. Colour-coded status chips show at a glance how far along each job is.

The attention badge in the sidebar nav shows the count of overdue jobs — investigate these first.

🖼️
Screenshot: Jobs page with status groups and search bar
2
Job Detail — Progress Tracking

Click any job to open it. You can see every step, its station, who has claimed it, and whether it is done. If a step is stuck, you can reassign or manually mark it complete here.

Attendance Monitoring

3
Live Attendance Board

Navigate to Attendance → Live. See in real time who is clocked in, who is absent, and how long each person has been in. Useful for the morning stand-up.

🖼️
Screenshot: Attendance Live board
4
Attendance Report

Navigate to Attendance → Report. Filter by worker and date range to see hours worked, late arrivals, and absences. Use this for payroll.

5
Leave Inbox (Admin only)

Navigate to Admin → Leave Inbox. Approve or reject pending leave requests. The worker is notified on their next login.

Stock Monitoring & Periodic Stock-Take

6
Low-Stock Alerts

The attention badge on Work → Materials shows how many items are below their buffer stock level. Navigate to Materials to see the full list with current quantities, buffer levels, and reorder suggestions.

7
Periodic Stock-Take (recommended every 1–2 months)

Navigate to Work → Stock-Take. Go through every item in the list and physically count what is in the storeroom. Enter the count in the Counted field next to each item.

  • Items where your count matches the system show a green tick
  • Items where your count differs show an orange highlight with the delta (±)

The discrepancy panel at the top summarises all differences. When ready, click Save. Stock levels are corrected immediately.

🖼️
Screenshot: Stock-Take page with some orange discrepancies highlighted
✅
Best practice: Do the stock-take at the start of the day before production begins, so quantities are stable while you count.
8
Purchase Orders

Navigate to Work → Reorder Queue to see all items below buffer stock. Click Generate Draft Orders to automatically create purchase order drafts grouped by supplier. Review and send the orders to suppliers from Work → Purchase Orders.
